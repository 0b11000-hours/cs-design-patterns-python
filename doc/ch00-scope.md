# Scope of the book
This book is a collection of notes and exercises that I have written for my students at the University of Applied
Sciences in Bern. It is intended to be used as a reference book for the course "Programming Fundamentals" and as a guide
for the exercises that are part of the course. The book covers the following topics:


## Topics

- Python Refresher
- Data Structures
- Divide and Conquer
- Dynamic Programming
- Greedy Algorithms
- Graph Algorithms
- Sorting Algorithms
- Searching Algorithms
- String Algorithms
- Number Theory
- Combinatorial Algorithms
- Probability Theory
- Sliding Window
- Two Pointers
- Bit Manipulation
- Backtracking
